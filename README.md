# Nightpunk

## Overview
Nightpunk is set in a dark, futuristic cyberpunk world where technology controls society. Massive, neon-lit cities serve as the backdrop. The graphics are pixel-style. You, as the player, have to navigate through the endless streets of the city and kill as much as possible. But be careful—if you cross the line, there is no turning back.

## Gameplay Loop
- Create a character  
- Enter the first level  
- Defeat all enemies  
- Collect coins  
- Buy upgrades in the shop  
- Finish the level  

## Controls
- *A / D* – Move  
- *Q, X, C* – Use perks  
- *W* – Jump  
- *E* – Use item  
- *F* – Collect item  
- *ESC* – Pause  
- *Left Mouse Button* – Attack  

## Features
- Two levels  
- Item and perk shop  
- Tutorial  
- Character creation  
- Various ambient and event-based sounds
